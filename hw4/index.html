    Part 5: Cloth Sim

    Explain in your own words what is a shader program and how vertex and fragment shaders work together to create lighting and material effects.

  • A shader program is a type of computer program used in computer graphics to determine the rendering of objects in a 3D scene. It operates on the GPU and is responsible for calculating the color, texture, lighting, and other visual properties of each pixel on the screen.
  • The vertex shader handles geometric transformations which determines the vertex position and prepares data for the fragment shader, while the fragment shader computes the final appearance of each pixel based on lighting, textures, and other factors.

    • Explain the Blinn-Phong shading model in your own words. Show a screenshot of your Blinn-Phong shader outputting only the ambient component, a screen shot only outputting the diffuse component, a screen shot only outputting the specular component, and one using the entire Blinn-Phong model.

  • Ambient Component: The light appears everywhere in the scene, regardless of the direct illumination which prevents the complete darkness.
  • Diffuse Component: The light is scattered equally in all directions upon hitting a surface. It is responsible for the color of an object when it is illuminated by direct light sources.
  • Specular Component: Specular highlights are bright, shiny spots on a surface that occur when light reflects off it in a concentrated manner.
  • The Blinn-Phong shading model combines the three components above to compute the final color of each pixel on a surface, resulting in a realistic representation of how light interacts with objects in a 3D scene.
